Rediseño de app - Tus Finanzas.

Completo rediseño en la parte visual de la aplicación móvil y nueva funcionalidad.

Préambulo

El banco lanzó una nueva aplicación móvil para que sus usuarios puedan ver sus gastos mensuales y fomentar el ahorro. Luego de unos meses en el mercado, el equipo ha decidido contratar una consultora UX que los ayude a definir la dirección y evolución del producto.

MVP entregado

  • Sistema desarrollado en IOS.
  • Data de 6 meses.
  • User primario y secundario.
  • App es independiente de la principal.
  • Landing page.

User testing

La prueba del usuario es una técnica utilizada en el diseño de interacción centrada en el usuario para evaluar un producto probándolo en los usuarios. Esto puede verse como una práctica de uso insustituible, ya que proporciona información directa sobre cómo los usuarios reales utilizan el sistema. Así que decidimos usar esta técnica en nuestra aplicación con estas tareas:

  1. Registrarse con la huella.
  2. Ocultar saldo disponible de la tarjeta.
  3. Crear una nueva meta de ahorro para una maestría.
  4. Crear un nuevo presupuesto para la categoía alimentación.
  5. Ver lugar y hora de un consumo en tambo.

Resultados

El test con usuarios de la categoría gastos arrojó que les agradó mucho la gráfica donde se puede visualizar todos sus gastos divididos en categorías. Así también agregar un presupuesto a cada categoría, porque el usuario quería autogestionar sus gastos y ponerse un límite.
